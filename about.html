About SpendList

SpendList is a Personal Expense Analytics Dashboard developed as a web programming project. The system allows users to track daily expenses, analyze monthly spending, and evaluate savings goals through a simple and interactive interface.

Project Objective

The objective of SpendList is to promote financial awareness by enabling users to monitor daily spending, analyze monthly financial patterns, and calculate required savings adjustments. The system demonstrates the practical use of HTML, CSS, and JavaScript in building a real-world web application.
